31733: Make ERIC catch the frog (if possible)
Used by the routine at 31573. Adds the frog to ERIC's inventory and makes a sound effect if the frog is sitting (i.e. not airborne) at the spot in front of ERIC.
Input
A Frog's animatory state (with bit 7 reset)
Output
31733 CP 28 28: Is the frog sitting down?
31735 RET NZ Return if not (ERIC cannot catch an airborne frog)
31736 LD HL,32747 32747 holds the inventory flags
31739 SET 2,(HL) Signal: got the frog
31741 LD H,212 212=frog
31743 JP 29568 Print the inventory and make a sound effect
